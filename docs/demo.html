A beautiful and modern notification library - Interactive Demo

Basic Notification Types

new NoticeJs({ text: 'Message', type: 'success', position: 'topRight' }).show();

Notifications with Title

new NoticeJs({ title: 'Success', text: 'Your changes have been saved!', type: 'success' }).show();

Timeout Control

Control how long notifications stay visible

// 5 seconds
new NoticeJs({ text: 'Message', timeout: 5000 }).show();

// No auto-close
new NoticeJs({ text: 'Message', timeout: false }).show();

Different Positions

new NoticeJs({ text: 'Message', position: 'topLeft' }).show();
// Options: topLeft, topRight, bottomLeft, bottomRight

Progress Bar Indicator

new NoticeJs({ text: 'Loading...', progressBar: true, timeout: 3000 }).show();

CSS Animations

Works with Animate.css and custom animations

new NoticeJs({
  text: 'Animated notification',
  animation: {
    open: 'animated bounceInRight',
    close: 'animated bounceOutLeft'
  }
}).show();

Multiple Notifications

// Show multiple notifications at different positions
new NoticeJs({ text: 'First', position: 'topLeft' }).show();
new NoticeJs({ text: 'Second', position: 'topRight' }).show();

Callbacks

Open browser console (F12) to see callback logs

new NoticeJs({
  text: 'Notification with callbacks',
  callbacks: {
    onShow: () => console.log('Shown!'),
    onClose: () => console.log('Closed!')
  }
}).show();
